How I run a project

I'm a Project Coordinator at Rubico Tech, managing multiple client projects at once — sprint planning, QA coordination, stakeholder updates, and reporting, from kickoff to launch. I catch problems early, keep documentation tidy, and make sure developers, designers, and clients are always working from the same page.

Beyond the standard toolkit, I build AI-assisted workflows using Claude and MCP connectors to automate documentation and reporting — pulling data from Jira, Slack, Gmail, and Google Drive into consistent, ready-to-review outputs. It means less time assembling status reports and more time actually unblocking the team.
